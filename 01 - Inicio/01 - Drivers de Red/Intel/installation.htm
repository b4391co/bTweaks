Intel® Ethernet Network Connections Software and Drivers Installation

The driver download package for Microsoft* Windows* and Microsoft Windows Server* operating systems are separate from the Intel® PROSet software download package.

Microsoft* Windows* and Microsoft Windows Server* Driver Packages

Download the appropriate driver package for your system. If you are running a 32-bit operating system, download Wired_driver_XX.X_32.exe. If you are running a 64-bit operating sytem, download Wired_driver_XX.X_x64.exe. These are self-extracting archives.

When you run the driver package, it will extract the files to a temporary directory, run the installation wizard, and remove the temporary files when the installation is complete. All language files are embedded in this archive. You do not need to download an extra language pack.

Extracting the Files Without Installing

You can use a tool such as WinRAR, WinZip, or 7-Zip to extract the files without installing them.

Device Driver Plug and Play Update

After you extract the files from the archive, you can use them to do a device driver plug and play update or install.

  1. Start Windows Device Manager
  2. Right click on the NIC you want to update and select Update Driver.
  3. Follow the directions in the Hardware Update Wizard to select the drivers you extracted.

Intel® PROSet

You must have installed Intel Ethernet device drivers before you can install Intel® PROSet.

Download the appropriate software package for your system. If you are running a 32-bit operating system, download Wired_PROSet_XX.X_32.exe. If you are running a 64-bit operating sytem, download Wired_PROSet_XX.X_x64.exe. These are self-extracting archives.

When you run the software package, it will extract the files to a temporary directory, run the installation wizard, and remove the temporary files when the installation is complete. All language files are embedded in this archive. You do not need to download an extra language pack.

Extracting the Files Without Installing

You can use a tool such as WinRAR, WinZip, or 7-Zip to extract the files without installing them.

Installazione del software e dei driver di connessione alla rete Ethernet di Intel®

  3. Seguire le istruzioni di Aggiornamento guidato hardware per selezionare i driver estratti.

Instalação do Software de conexões de rede Ethernet Intel® e drivers

O pacote de download do driver para os sistemas operacionais Microsoft* Windows* e Microsoft Windows Server* são separados do pacote de download do software Intel® PROSet.

Pacotes dos drivers Microsoft* Windows* e Microsoft Windows Server*

Baixe o pacote de driver adequado para o seu sistema. Se você estiver executando um sistema operacional de 32 bits, baixe Wired_driver_XX.X_32.exe. Se você estiver executando um sistema operacional de 64 bits, baixe Wired_driver_XX.X_x64.exe. Estes arquivos são de extração automática.

Todos os arquivos de idioma são incorporados a esse arquivamento. Não é necessário baixar um pacote de idioma extra.

  2. Clique com o botão direito na placa de rede a ser atualizada e selecione 'Atualizar driver'.
  3. Siga as instruções do Assistente para atualização de hardware, para selecionar os drivers que você extraiu.

Intel® PROSet

您必須先安裝 Intel 乙太網路裝置驅動程式，才能安裝 Intel® PROSet。

為系統下載正確的軟體套件。如果執行的是 32 位元作業系統，請下載 Wired_PROSet_XX.X_32.exe。如果執行的是 64 位元作業系統，請下載 Wired_PROSet_XX.X_x64.exe。這些封存可自動解壓縮。

執行軟體套件時，它會將檔案解壓縮至暫存目錄、執行安裝精靈，並且在安裝完成時移除暫存檔案。所有的語言檔案都內嵌在這個封存檔中。您不需下載額外的語言套件。

擷取檔案但不安裝

您無須安裝，即可使用 WinRAR、WinZip 或 7-Zip 這類工具解壓縮檔案。

インテル® イーサネット・ネットワーク・コネクションのソフトウェアとドライバーのインストール

Microsoft* Windows* および Microsoft* Windows Server* オペレーティング・システム用ドライバー・ダウンロード・パッケージは、インテル® PROSet ソフトウェアのダウンロード・パッケージとは分離されています。

Microsoft* Windows* および Microsoft* Windows Server* ドライバーパッケージ

システムに適したドライバーパッケージをダウンロードします。32 ビットのオペレーティング・システムを実行している場合は、Wired_driver_XX.X_32.exe をダウンロードしてください。64 ビットのオペレーティング・システムを実行している場合は、Wired_driver_XX.X_x64.exe をダウンロードしてください。これらは自己展開アーカイブです。

ドライバーパッケージを実行すると、ファイルが展開されて一時ディレクトリーに保存され、インストール・ウィザードを実行します。インストールが終わったら、この一時ファイルを削除します。言語ファイルはすべてこのアーカイブに組み込まれています。言語パックを別途ダウンロードする必要はありません。

インストールせずに展開する

WinRAR、WinZip、または 7-Zip などのツールを利用すると、インストールせずにファイルを展開することができます。

デバイスドライバーのプラグ・アンド・プレイ・アップデート

アーカイブから展開したファイルを使用して、デバイスドライバーのプラグ・アンド・プレイ・アップデートまたはインストールができます。

  1. Windows* のデバイス マネージャーを起動する
  2. アップデートしたい NIC を右クリックして、[ドライバーの更新] を選択します。
  3. ハードウェアの更新ウィザードの指示に従って、展開したドライバーを選択します。

インテル® PROSet

インテル® PROSet をインストールする前に、インテル® イーサネットのデバイスドライバーをインストールしておく必要があります。

システムに適したソフトウェア・パッケージをダウンロードします。32 ビットのオペレーティング・システムを実行している場合は、Wired_PROSet_XX.X_32.exe をダウンロードしてください。64 ビットのオペレーティング・システムを実行している場合は、Wired_PROSet_XX.X_x64.exe をダウンロードしてください。これらは自己展開アーカイブです。

ソフトウェア・パッケージを実行すると、ファイルは展開されて一時ディレクトリーに保存され、インストール・ウィザードを実行します。インストールが終わったら、この一時ファイルを削除します。言語ファイルはすべてこのアーカイブに組み込まれています。言語パックを別途ダウンロードする必要はありません。
